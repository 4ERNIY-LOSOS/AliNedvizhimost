# AliNedvizhimost

Это десктопное WPF-приложение, представляющее собой платформу для продажи и покупки недвижимости. Приложение разработано на .NET с использованием паттерна MVVM и использует базу данных SQL Server для хранения данных.

## Настройка базы данных

Перед первым запуском приложения необходимо создать и заполнить базу данных. Для этого у вас должен быть установлен SQL Server (например, бесплатная версия Express Edition).

1.  Откройте терминал (Command Prompt или PowerShell).
2.  Убедитесь, что вы находитесь в корневой папке проекта.
3.  Выполните следующие команды по очереди:

    ```shell
    # 1. Создание структуры базы данных и таблиц
    # Эта команда создает базу данных AliNedvizhimost и все необходимые таблицы.
    sqlcmd -S localhost\SQLEXPRESS -i setup_database.sql

    # 2. Заполнение базы данных тестовыми данными
    # Эта команда добавляет тестовых пользователей и объявления.
    # Флаг -f 65001 важен для корректной обработки русских символов в скрипте.
    sqlcmd -S localhost\SQLEXPRESS -i seed_database.sql -f 65001
    ```

    **Примечание:** Если ваш экземпляр SQL Server называется иначе, чем `localhost\SQLEXPRESS`, замените это значение на правильное имя вашего сервера.

## Запуск и разработка

Вы можете запустить и разрабатывать этот проект, используя Visual Studio или Visual Studio Code.

### В Visual Studio (Рекомендуется)

1.  **Клонируйте репозиторий:**
    ```shell
    git clone https://github.com/4ERNIY-LOSOS/AliNedvizhimost.git
    cd AliNedvizhimost
    ```
2.  **Откройте решение:** Откройте файл `AliNedvizhimost.sln` в Visual Studio 2022 или новее.
3.  **Настройте базу данных:** Перед первым запуском выполните шаги из раздела "Настройка базы данных".
4.  **Запустите проект:** Нажмите `F5` или зеленую кнопку "Start" на панели инструментов, чтобы скомпилировать и запустить приложение. Visual Studio автоматически восстановит все необходимые NuGet пакеты.

### В Visual Studio Code

Запуск WPF-проектов в VS Code требует предварительной настройки.

1.  **Установите зависимости:**
    *   [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)
    *   Расширение [C# Dev Kit](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csdevkit) для VS Code.
2.  **Клонируйте репозиторий** и откройте папку проекта в VS Code.
3.  **Настройте базу данных:** Выполните шаги из раздела "Настройка базы данных", используя встроенный терминал в VS Code (`Ctrl+``).
4.  **Сборка и запуск:**
    *   Откройте встроенный терминал.
    *   Выполните команду для сборки проекта:
        ```shell
        dotnet build
        ```
    *   После успешной сборки выполните команду для запуска приложения:
        ```shell
        dotnet run --project AliNedvizhimostApp
        ```

## Основные возможности

-   **Аутентификация:** Регистрация и вход для пользователей с ролями "Клиент" и "Администратор".
-   **Каталог недвижимости:** Просмотр списка объектов недвижимости с интерактивной фильтрацией (по цене, площади, комнатам) и сортировкой.
-   **Управление объявлениями:** Пользователи могут добавлять, редактировать и удалять собственные объявления.
-   **Система сообщений:** Возможность связаться с продавцом по поводу объявления и вести переписку.
-   **Панель администратора:** Инструменты для администратора для управления всеми пользователями и объявлениями в системе.
